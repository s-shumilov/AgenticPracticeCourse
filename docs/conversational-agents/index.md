# Conversational Agents

**Build a conversational AI agent grounded in UiPath Security documentation, extend it with tools, then put it through its paces with real conversations.**

## Overview

**Agent Builder** lets you create conversational agents that respond to questions in real time, drawing on grounded knowledge sources and connected tools. This exercise takes you through the full lifecycle: creating an agent, exploring the tools it can use, and testing how it holds up across a range of real-world conversations.

| Step | Focus |
| ---: | :--- |
| [**Create a Conversational Agent**](1-create-agent.md) | Build an agent grounded in UiPath Security documentation using context grounding |
| [**Understanding Available Tools**](2-review-tools.md) | Explore the tools your agent can use and how they process data |
| [**Testing Conversations**](3-testing-conversations.md) | Run conversations from different angles and evaluate the quality of responses |
